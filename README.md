# atividade1PP
Demonstração dos Códigos das Primeiras Aulas de Projeto de Programas

// 1º O primeiro arquivo, que é aula01.js, contém  os comandos básicos a serem executados na linguagem Java Script,  na qual são abordados:
    Estruturas Sequenciais: Atribui-se uma variável  com a palavra let antes do nome da variável, e depois executa-se a sáida da palavra através do comando console.log();
    Estruturas Condicionais: Utiliza-se o comando if e else, com o intuito de saber qual a saída correta para a variável no comando  console.log();
    Estruturas Iterativas: Utiliza-se o comando for para determinar quantas vezes uma determinada variável irá se repetir na operação, com a definição de um iterador i que
    iniciará com o valor zero e irá até determinado limite, sendo iterado na variação de dois passos;
    Estruturas  Container: Foi atribuída uma variável objeto, a qual armazen-ase em uma lista o nome de uma pessoa e a sua idade, e através do comando console.log() executa-se
    as variáveis armamzenadas por essa estrutura.
    
// 2º Os Demais arquivos são aqueles ministrados na segunda aula de projetos de programas, os quais  foram instalados com o intuito de executar o progrma em uma pasta .ts:
    *O primeiro passo foi criar um folder de nome aula02, onde todos esse arquivos estariam armazenados;
    * O segundo passo foi utilizar os comandos node -v e npm -v no terminal do programa, afim de que sejam transmitidos os dados da versão do progrma do node js;
    *  O  terceiro passo foi o de utilizar o comando cd .\aula02\ e o comando npm init, com o intuito de configurar alguns dados básicos do programa, o que levou a instalação 
    do pacote  package.json;
    * O quarto passo foi o  uso do comando npm install typescript  -D, no temrnial, com o intuito de instalar os pacotes contidos dentro do folder node e também com a finalidade
    de colocar a configuração devDependecies dentro do pacote package.json;
    * O quinto passo foi a execução do comando  tsc --init dentro do terminal, com o intuito de instalar a pasta de configurações do typescritp denominada:  tsconfgi.json
    * O sétimo passo foi definir as configurções fundamenta  dentro da pasta tsconfig.json como:  lib,  outDir, target e module. A definição dessas configurações foi tirar do
    modo de comentário e atribuindo em forma de string os seus devidos atributos dentro dessa pasta;
    *O  penúltimo passo foi definir dentro da estrutura "scripts", da pasta  package.json,  a atribuição da variável start com o intuito de que o programa funcionasse na pasta
    index.ts;
    *O último passo foi executar o comando no terminal atribuindo a uma variável let um tipo string e  a saída do comando sendo realizada através do comandfo console.log.
